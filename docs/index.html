常见问题

你可能想问的

企业账号能用吗？

能。使用 OAuth2 浏览器授权，腾讯会议不区分个人/企业，只要账号能登录就能授权。

登录超时怎么办？

不要加 `--no-browser` 参数。让 CLI 自动弹出浏览器窗口，浏览器中有完整的登录和授权流程，不容易超时。

发邮件需要额外装什么？

需要 agently-cli（发邮件）和 react-email（生成精美邮件）。创建会议后如果选择发邀请，AI 会自动检查并引导安装。

如何卸载？

运行 `npm uninstall -g @tencentcloud/tmeet` 并删除 `~/.claude/skills/SKILL.md` 即可。
