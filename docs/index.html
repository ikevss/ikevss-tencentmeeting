推荐安装方式（不需要懂命令行）

安装 Claude Code 后，直接对 AI 说下面的话就行

第 1 步 · 对 AI 说
帮我安装腾讯会议技能 ikevss-tencentmeeting
Claude Code 会自动从 GitHub 下载、安装并配置好一切。
第 2 步 · 登录（30 秒，只需一次）
帮我登录腾讯会议
浏览器自动弹出，扫码或输入密码授权。之后永久免登录，不用记密码。
第 3 步 · 直接用
帮我创建一个明天下午3点的腾讯会议，讨论Q2财报
就像跟助理说话一样。建会后会自动问你要不要发邮件通知参会人。
或者手动安装（适合习惯命令行的用户）

前提条件：电脑已安装 Node.js（版本 ≥ 16）。打开终端（Windows 按 Win+R 输入 cmd，Mac 搜索"终端"），逐行粘贴运行：

# 1. 安装腾讯会议命令行工具
npm install -g @tencentcloud/tmeet@v1.0.12 # 2. 下载并安装技能
npx https://github.com/ikevss/ikevss-tencentmeeting # 3. 登录腾讯会议
tmeet auth login # 4. 重启 Claude Code 即可使用

⚠️ 手动安装完成后需要重启 Claude Code 使技能生效。

常见问题

你可能想问的

完全不懂命令行能用吗？

能。推荐安装方式不需要你敲任何命令——直接对 Claude Code 说"帮我安装腾讯会议技能"就行，AI 会自动完成所有配置。

企业账号能用吗？

能。使用 OAuth2 浏览器授权，腾讯会议不区分个人/企业，只要账号能登录就能授权。

安装后在哪用？

在 Claude Code 对话窗口里直接用。跟它说"帮我建一个明天下午3点的会"，它就帮你创建。不需要打开腾讯会议客户端。

需要额外装什么软件吗？

只需要 Claude Code（AI 编程助手）。发邮件通知需要 agently-cli（创建会议后 AI 会自动引导安装）。

登录超时怎么办？

不要勾选"无浏览器模式"。让浏览器自动弹出，浏览器中有完整的登录和授权流程，不容易超时。

如何卸载？

对 AI 说"帮我卸载腾讯会议技能"，或手动删除 ~/.claude/skills/SKILL.md 文件。
