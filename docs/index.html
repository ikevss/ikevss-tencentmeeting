四环节旅程

从建会到复盘，四步走完

不是四个工具，是一个对话框。一句话下去，AI 自动串联建会、邀请、通知、复盘全流程。

1建会

一句话创建，支持周期会议、成员增删替换。对 AI 说"帮我建一个明天下午3点的会，讨论Q2财报"——拿到会议号和入会链接。

✨ 创建完自动问："需要生成邀请函吗？"
tmeet meeting create
subject: "Q2财报讨论"
start: 2026-07-23T15:00+08:00
end: 2026-07-23T16:00+08:00
→ meeting_code: 192 715 206

2邀请函

自动生成精美 HTML 邀请函，包含会议主题、时间、议程、入会方式。桌面端和手机端都能正常看。浏览器预览确认后再发出去。

3发送通知

通过 agently-cli 发送邮件通知参会人，附带 ICS 日历附件。收件人双击即可把会议加进日历，不用手动录入时间。

邮件 + ICS 日历附件，一键加进日程
agently mail send
to: ["zhangsan@company.com", "lisi@company.com"]
subject: "会议邀请：Q2财报讨论"
attachment: meeting.ics
→ 已发送 2 封邮件

常见问题

企业账号真的能用吗？

可以。本技能使用 OAuth2 浏览器授权，腾讯会议不区分个人/企业，只要账号能登录就能授权。

SSH 服务器没有浏览器怎么办？

用 tmeet auth login --no-browser，CLI 会打印一个授权链接。在你自己的电脑浏览器打开完成授权即可。

发邮件需要额外装什么？

需要 agently-cli（发邮件）和 react-email（生成精美邮件）。创建会议后如果选择发邀请，AI 会自动检查并引导安装。

支持哪些平台？

macOS、Linux、Windows 全支持。Node.js >= 16 就行。

凭证安全吗？

AES-256-GCM 加密存储，明文不落盘。OAuth2 流程让你的密码不会暴露给 CLI。随时可以 tmeet auth logout 清除。

如何卸载？

运行 npm uninstall -g @tencentcloud/tmeet 并删除 ~/.claude/skills/SKILL.md 即可。
